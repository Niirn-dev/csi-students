\section{Реализация регулятора}
Для реализации регулятора, представим его передаточную функцию в виде произведения типовых звеньев, для которых уже разработаны электрические схемы\cite{besekersky} и соединим их последовательно. Так как коэффициент $q$ не сильно меньше 1, пренебрежём им и получим передаточную функцию регулятора следующего вида:
\begin{align}
	W_p(s) = \frac{K_p(T_2s + 1)(T_1s + 1)^2}{s(T_3s + 1)^2}.
\end{align}

Разбив на множители, получим
\begin{align}
	W_p(s) = \frac{T_2s + 1}{T_0s}\cdot\sqrt{K_p}\frac{T_1s + 1}{T_3s + 1}\cdot\sqrt{K_p}\frac{T_1s + 1}{T_3s + 1}.
\end{align}

Тогда электрическая схема регулятора принет вид, представленный на рисунке \ref{realisationScheme}.
\begin{figure}[ht!]
	\centering
	\includegraphics[width = 0.9\textwidth]{realisation/scheme}
	\caption{Электрическая схема реализации регулятора}
	\label{realisationScheme}
\end{figure}

На представленной схеме выполняются равенства
\begin{align}
	\begin{cases}
		R_3 = R_5\\
		R_4 = R_6\\
		C_2 = C_3
	\end{cases}
\end{align}

Рассчитаем значения элементов схемы.
\begin{align}
	\begin{cases}
		\begin{cases}
			T_0 = R_1C_1 = 1\\
			T_2 = R_2C_1 = 0,222
		\end{cases}\\
		\begin{cases}
			T_1 = R_3C_2 = 0,11\\
			T_3 = \frac{R_3R_4}{R_3 + R_4}C_2 = 0,0479\\
			\sqrt{K_p} = \frac{R_4}{R_3 + R_4} = 0,5348
		\end{cases}
	\end{cases}
\end{align}

Как можно заметить, в первой системе 2 уравнения и 3 неизвестных, поэтому решим эту систему, приняв $R_2 = 1$ Ом. Тогда получим
\begin{align}
	\begin{cases}
		\begin{cases}
			R_2 = 1 \text{ Ом}\\
			C_1 = \frac{0,222}{1} = 0,222 \text{ Ф}\\
			R_1 = \frac{1}{0,222} = 4,505 \text{ Ом}
		\end{cases}\\
		\begin{cases}
			C_2 = \frac{0,11}{R_3}\\
			R_4 = 1,1496R_3\\
			\frac{R_6\cdot1,1496R_6}{R_6 + 1,1496R_6}\cdot\frac{0,11}{R_6} = 0,0479
		\end{cases}
	\end{cases}
\end{align}

Решив вторую систему получим следующие значения для всех элементов схемы:
\begin{align}
	\begin{cases}
		R_1 = 4,505 \text{ Ом}\\
		R_2 = 1,000 \text{ Ом}\\
		R_3 = R_5 = 0,814 \text{ Ом}\\
		R_4 = R_6 = 0,936 \text{ Ом}\\
		C_1 = 0,222 \text{ Ф}\\
		C_2 = C_3 = 0,130 \text{ Ф}
	\end{cases}
\end{align}
